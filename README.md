# rock-paper-scissors
Rock Paper Scissors is a simple and interactive web game built using HTML, CSS, and JavaScript. The player competes against the computer, which generates random moves. The project demonstrates basic game logic, DOM manipulation, and event handling.

# Features

 * Player vs Computer gameplay

 * Random computer choice generation

 * Instant result display (Win / Lose / Draw)

 * Clean and user-friendly interface

 * Responsive design for different screen sizes

# Technologies Used

 * HTML – Structure of the game

 * CSS – Styling and layout

 * JavaScript – Game logic and interactivity

# How to Play

 * Open the game in your browser.

 * Choose Rock, Paper, or Scissors.

 * The computer makes a random choice.

 * The winner is decided based on classic game rules.

 * The result is displayed instantly.
